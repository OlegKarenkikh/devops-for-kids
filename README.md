## 🗺️ Дорожная карта курса

```
Старт → Терминал → Git → GitHub → Docker → Docker Compose → Kubernetes → 🏆
```

| Модуль | Уроки | Что узнаешь |
|--------|-------|-------------|
| 🐧 [Терминал и Linux](lessons/module1-terminal/) | 1–5 | Команды bash, файлы, права |
| 🌿 [Git и GitHub](lessons/module2-git/) | 6–8 | Контроль версий, совместная работа |
| 🐳 [Docker](lessons/module3-docker/) | 9–11 | Контейнеры, образы, Dockerfile |
| 🎼 [Docker Compose](lessons/module4-compose/) | 12–13 | Несколько сервисов вместе |
| ⚙️ [Kubernetes](lessons/module5-kubernetes/) | 14–20 | Оркестрация, масштабирование |

---

## 🚀 Быстрый старт

**Что нужно:**
- Компьютер с Linux (Ubuntu 22.04+) или WSL на Windows
- Доступ к терминалу
- ~4 часа свободного времени на весь курс

**Начать с урока 1:**
```bash
# Просто открой терминал и напиши:
whoami
```
