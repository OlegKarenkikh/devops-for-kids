## 🚀 Быстрый старт

**Что нужно:**
- Компьютер с Linux (Ubuntu 22.04+) или WSL на Windows
- Доступ к терминалу
- ~4 часа на весь курс

```bash
# Открой терминал и напиши первую команду:
whoami
```
