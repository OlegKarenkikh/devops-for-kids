# 🎨 Иллюстрация: module2-pat-token

## Назначение
Объяснить новичку что такое PAT (Personal Access Token) в GitHub.

## Содержание иллюстрации

### Схема (слева направо):
1. **Пользователь** (иконка человека) → открывает github.com/settings/tokens
2. **Форма создания токена** (упрощённо):
   - Название: `my-script`
   - Срок: 90 дней
   - Права: ✅ repo
3. **Токен** (строка `ghp_XXXX...`) с иконкой 🔑
4. **Применение**:
   - В терминале: `export GITHUB_TOKEN="ghp_..."`
   - В git remote URL: `https://TOKEN@github.com/...`
   - В curl: `-H "Authorization: Bearer TOKEN"`

### Визуальные акценты:
- ⚠️ "Копируй сразу — показывается только один раз!"
- 🗓️ "Токен живёт N дней — потом нужно создать новый"
- ❌ Зачёркнутый пароль → ✅ Токен

## Стиль
Дружелюбный, как остальные иллюстрации курса. Тёмный фон с яркими метками.
