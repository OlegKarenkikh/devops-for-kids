# 🎨 Иллюстрация: module6-sqlite-table

## Назначение
Показать как SQLite хранит данные в таблице и как SQL запросы работают с ней.

## Схема
```
Файл items.db
┌────┬──────────┬───────┐
│ id │ name     │ emoji │   ← CREATE TABLE
├────┼──────────┼───────┤
│  1 │ Гитара   │  🎸   │   ← INSERT
│  2 │ Книга    │  📚   │   ← INSERT
│  3 │ Кот      │  🐱   │   ← INSERT
└────┴──────────┴───────┘
        ↑
  SELECT * FROM items → [все строки]
  INSERT INTO items (name, emoji) VALUES (?, ?)
  DELETE FROM items WHERE id = 2
```

### Ключевые элементы:
- Таблица с тремя строками и заголовками
- Стрелки с SQL командами (INSERT/SELECT/DELETE)
- Файловая иконка: "items.db — один файл на диске"
- Надпись: "SQLite = база данных без сервера"

## Стиль
Таблица в центре, команды вокруг со стрелками. Светлый фон таблицы.
