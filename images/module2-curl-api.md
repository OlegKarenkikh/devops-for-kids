# 🎨 Иллюстрация: module2-curl-api

## Назначение
Показать как curl работает как «терминальный браузер» к GitHub API.

## Схема иллюстрации

```
Терминал                      Интернет              GitHub API
   │                              │                      │
   │  curl -H "Authorization:..." │                      │
   │─────────────────────────────>│─────────────────────>│
   │                              │                      │
   │       JSON ответ             │   {"login": "user"}  │
   │<─────────────────────────────│<─────────────────────│
   │                              │                      │
```

### Блоки в иллюстрации:
1. **Терминал** (чёрное окно) — команда `curl`
2. **Стрелка →** с надписью "HTTP GET запрос + Bearer Token"
3. **GitHub API** (логотип GitHub) — иконка сервера
4. **Стрелка ←** с надписью "JSON ответ"
5. **Результат** — форматированный JSON в терминале

### Примеры команд (вставить как код):
- `curl .../user` → Профиль пользователя
- `curl .../repos` → Список репозиториев
- `curl -X POST .../repos` → Создать репозиторий
