## 📖 Урок 1 — Что такое терминал?

### 🧠 Объяснение

Представь, что твой компьютер — это умный пёс. Обычно ты командуешь ему мышкой. Но **терминал** — это способ говорить с ним текстом: написал команду — компьютер выполнил.

### 🖥️ Как открыть

- **Ubuntu:** `Ctrl + Alt + T`
- **macOS:** `Cmd + Пробел` → `Terminal`
- **Windows (WSL):** найди «Ubuntu» в меню пуск

### 💻 Первые команды

```bash
whoami        # Кто я? Узнай своё имя пользователя
pwd           # Где я? Покажи текущую папку
date          # Который час?
```

### 🧪 Задание 1

1. Открой терминал
2. Напиши `whoami` → Enter
3. Напиши `pwd` → Enter
4. Напиши `date` → Enter

> 💡 Если ошибся — нажми `Ctrl+C` (кнопка «Стоп»)

### 💻 Навигация

```bash
ls            # Что здесь? Список файлов
ls -la        # Подробно + скрытые файлы
cd Documents  # Войти в папку
cd ..         # Выйти на уровень выше
cd ~          # Перейти домой
clear         # Очистить экран
```

### 💻 Файлы и папки

```bash
mkdir мой-проект          # Создать папку
touch файл.txt            # Создать пустой файл
echo "Привет!" > файл.txt # Записать текст
echo "Ещё!" >> файл.txt   # Добавить в конец
cat файл.txt              # Прочитать файл
cp файл.txt копия.txt     # Скопировать
mv копия.txt папка/       # Переместить
rm файл.txt               # Удалить ⚠️
```

> ⚠️ `rm` удаляет навсегда — корзины нет!

## 📖 Урок 4 — Поиск и текст

```bash
grep "банан" список.txt   # Найти слово
find . -name "*.txt"      # Найти файлы
head -n 3 файл.txt        # Первые 3 строки
tail -n 3 файл.txt        # Последние 3 строки
wc -l файл.txt            # Количество строк
cat файл.txt | grep "а" | sort   # Конвейер команд
```

### 🧪 Задание 4

```bash
echo "манго"  >> список.txt
echo "банан"  >> список.txt
grep "банан" список.txt   # Сколько раз?
wc -l список.txt
```

## 📖 Урок 5 — Права доступа

</div>

### 🔍 Читаем права

```
-rw-r--r-- 1 ivan ivan  42 Mar 23 файл.txt
 │││└── остальные: r-- (только читать)
 ││└─── группа: r-- (только читать)
 │└──── ты: rw- (читать + писать)
 └───── тип: - файл, d папка
```

| Символ | Значение |
|--------|---------|
| `r` | read — читать |
| `w` | write — писать |
| `x` | execute — запускать |
| `-` | нет права |

### 💻 Команды

```bash
chmod +x скрипт.sh    # Добавить право запуска
chmod 644 файл.txt    # rw-r--r-- (стандарт)
chmod 755 скрипт.sh   # rwxr-xr-x (исполняемый)
chmod 600 секрет.txt  # rw------- (только ты)
sudo команда          # От имени администратора
```

### 🧪 Задание 5

```bash
touch секрет.txt
ls -l секрет.txt    # Посмотри права
chmod 600 секрет.txt
ls -l секрет.txt    # Что изменилось?
```
