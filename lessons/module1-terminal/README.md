## 📖 Урок 1 — Что такое терминал?

### 🧠 Объяснение

Представь, что твой компьютер — это умный пёс. Обычно ты командуешь ему мышкой. Но **терминал** — это способ говорить с ним текстом. Ты пишешь команду — компьютер выполняет.

### 🖥️ Как открыть терминал

- **Ubuntu:** `Ctrl + Alt + T`
- **macOS:** `Cmd + Пробел` → напиши `Terminal`
- **Windows (WSL):** найди «Ubuntu» в меню

### 💻 Первые команды

```bash
whoami        # Кто я? Узнай своё имя пользователя
pwd           # Где я? Покажи текущую папку
date          # Сколько времени?
```

### 🧪 Задание

1. Открой терминал
2. Напиши `whoami` — нажми Enter
3. Напиши `pwd` — нажми Enter
4. Напиши `date` — нажми Enter

> 💡 Если ошибся — нажми `Ctrl+C` (кнопка «Стоп»)

## 📖 Урок 2 — Смотрим вокруг

### 💻 Команды

```bash
ls            # Что здесь? Список файлов и папок
ls -l         # То же, но подробно
ls -la        # Подробно + скрытые файлы
cd Documents  # Войти в папку Documents
cd ..         # Выйти на уровень выше
cd ~          # Перейти домой
clear         # Очистить экран
```

### 📊 Структура файловой системы

```
/ (корень)
├── home/
│   └── ivan/        ← Твоя домашняя папка (~)
│       ├── Documents/
│       ├── Downloads/
│       └── Pictures/
├── etc/             ← Настройки системы
├── var/             ← Логи и данные
└── usr/             ← Программы
```

### 🧪 Задание

```bash
ls
cd Documents
pwd
ls
cd ..
pwd
```

## 📖 Урок 3 — Файлы и папки

### 💻 Команды

```bash
mkdir мой-проект          # Создать папку
touch файл.txt            # Создать пустой файл
echo "Привет!" > файл.txt # Записать текст в файл
echo "Ещё!" >> файл.txt   # Добавить в конец файла
cat файл.txt              # Прочитать файл
cp файл.txt копия.txt     # Скопировать
mv копия.txt папка/       # Переместить/переименовать
rm файл.txt               # Удалить файл ⚠️
rmdir пустая-папка/       # Удалить пустую папку
```

> ⚠️ **Осторожно:** `rm` удаляет файлы навсегда, корзины нет!

### 🧪 Задание

```bash
mkdir урок3
cd урок3
touch список.txt
echo "яблоко" > список.txt
echo "банан" >> список.txt
echo "апельсин" >> список.txt
cat список.txt
```

## 📖 Урок 4 — Поиск и работа с текстом

### 💻 Команды

```bash
grep "банан" список.txt       # Найти слово в файле
grep -r "слово" папка/        # Найти в папке рекурсивно
find . -name "*.txt"          # Найти файлы по маске
find / -name "passwd" 2>/dev/null
head -n 5 файл.txt            # Первые 5 строк
tail -n 5 файл.txt            # Последние 5 строк
wc -l файл.txt                # Количество строк
sort файл.txt                 # Отсортировать строки
uniq файл.txt                 # Убрать дубликаты
```

### 💡 Конвейер команд (pipe)

```bash
# Выводи результат одной команды в другую
cat список.txt | grep "а" | sort
```

### 🧪 Задание

```bash
echo "манго" >> список.txt
echo "банан" >> список.txt
grep "банан" список.txt   # Сколько раз встречается?
wc -l список.txt
sort список.txt
```

## 📖 Урок 5 — Права доступа

### 🧠 Объяснение

Каждый файл имеет три группы прав:
- **u** (user) — ты сам
- **g** (group) — твоя группа
- **o** (other) — все остальные

### 📊 Расшифровка `ls -l`

```
-rw-r--r-- 1 ivan ivan 42 Mar 23 привет.txt
 ^^^            
 |||— остальные: r (только читать)
 ||— группа: r (только читать)  
 |— ты: rw (читать + писать)
```

| Символ | Значение |
|--------|----------|
| `r` | read — читать |
| `w` | write — писать |
| `x` | execute — запускать |
| `-` | нет права |

### 💻 Команды

```bash
chmod +x скрипт.sh       # Добавить право запуска
chmod 644 файл.txt       # rw-r--r-- (стандарт)
chmod 755 скрипт.sh      # rwxr-xr-x (исполняемый)
chmod 600 секрет.txt     # rw------- (только ты)
sudo команда             # Запустить как администратор
```

### 🧪 Задание

```bash
touch секрет.txt
ls -l секрет.txt
chmod 600 секрет.txt
ls -l секрет.txt
# Что изменилось?
```
